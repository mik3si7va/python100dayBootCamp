# 100 Days of Python 🐍

A personal journey through Angela Yu's 100 Days of Python course.

This repository serves as both a learning log and a coding diary, documenting my progression from beginner Python projects to more advanced topics such as OOP, automation, data analysis, and web development.

Each folder represents a day of the challenge and contains the code, experiments, successes, mistakes, and lessons learned along the way.
